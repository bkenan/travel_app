Travel API test

This HTML/JS page tests the Travel API by listing all; then creating, reading, updating, and deleting a destination using the REST API. A few notes:

  • API endpoints, property names and values for the travel app are hardcoded in the client-side /javascript/api.js file linked in this HTML page.
  • To test your own API you will have to modify these values.
  • See the console for output as the process completes.
